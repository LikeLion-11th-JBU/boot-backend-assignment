인터넷 통신의 규칙

HTTP 통신과 URL

일정한 규칙을 정해서 보내진 요청을 일정한 규칙으로 응답을 함

HyperText Transfer Protocal = HTTP 하이퍼 텍스트 전송 규칙

protocol == 규칙

HyperText == 다른 리소스로 이동할 수 있는 링크를 뜻함


URL Uniform Resource Locator

https://www.google.com/search?q=techit

https:// = 프로토콜 -통신 규칙, HTTP(HTTPS),FTP등

www.google.com = 호스트 - 서버의 주소, google.com을 호스트 네임이라 지칭

/search = 경로(path) - 호스트 내 서비스의 위치, 서비스 별로 분할 ex) 검색, 회원 등

?q=techit == 쿼리 문자열(query String) - ?기호로 시작, &로 연결, 키/값 쌍으로 구성 
사용자 임의로 쿼리 문자를 사용할 수 없음
