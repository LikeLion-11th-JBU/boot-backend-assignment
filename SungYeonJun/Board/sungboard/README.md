
# 게시판 만들기
`게시판` 웹 애플리케이션
`Swagger`을 이용해보기 위한 웹 애플리케이션 프로젝트 입니다.

# 버전
###  Backend
| 기술 & 환경       | 버전     |
|---------------|--------|
| Spring Boot   | 2.7.13 |
| Gradle        | 8.1.1  |
| Mysql         | 8.0.32 |
| Java          |   11   |
| IntelliJ IDEA | 23.1.3 |
| Swagger       | 3.0.0  |

# 구현 기능
* 게시판 기능
    * 게시글 작성
    * 게시글 수정
    * 게시글 삭제
* 페이징 기능
    * 페이지 넘어가기
    * 페이지 돌아가기
    * 최대 `10개`페이지 모아서 보기

# API 명세서
* Swagger 3.0 ver
* 스웨거 진입점 `http://localhost:8080/swagger-ui/index.html`
